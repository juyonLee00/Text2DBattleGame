## 규칙
### 1) git, branch 규칙
- PR올릴 때 기능에 대한 코멘트 작성하기
- branch 명명: feature/유저명/기능명


<br/>

### 3) Commit Convention
- [type] body
- type
    - feat : 새로운 파일 생성, 새로운 기능 추가
    - fix : 버그 수정
    - refactor : 코드 리팩토링
    - chore : 동작에 영향을 주지 않는 사항들 (주석, 정렬 등등)
- body : 구현한 기능 설명
